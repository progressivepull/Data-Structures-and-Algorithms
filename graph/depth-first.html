Depth-First Search (DFS)

Depth-First Search (DFS) is a graph traversal algorithm that explores as far as possible along one branch before backtracking. Starting from a chosen source node, DFS dives deep to an unvisited neighbor, then to that neighbor’s neighbor, and so on, only retreating when it reaches a node with no new neighbors to visit.

How It Works

The process follows a clear sequence:

  1. Initialize: Choose a starting node and mark it as discovered.
  2. Use a Stack (or Recursion): Push the start node onto a stack (LIFO order), or use the call stack recursively.
  3. Process: Repeat until the stack is empty:
    • Peek the top node on the stack (the current node).
    • If it has an unvisited neighbor, select one, mark it discovered, and push it to the stack.
    • If no unvisited neighbors remain, pop the current node (it’s now fully explored).
  4. End: When the stack is empty, all reachable nodes have been explored.

Example

Consider the following simple graph (represented as adjacency lists):

A: B, C
B: D, E
C: F
D: —
E: —
F: —

If we start DFS from node A (choosing neighbors in alphabetical order):

  • Stack = [A]
  • From A, go to B → Stack = [A, B]
  • From B, go to D → Stack = [A, B, D]; D has no new neighbors → pop D → Stack = [A, B]
  • From B, next neighbor E → Stack = [A, B, E]; E done → pop E → Stack = [A, B]; B done → pop B → Stack = [A]
  • From A, next neighbor C → Stack = [A, C]; from C, go to F → Stack = [A, C, F]; F done → pop F → [A, C]; C done → pop C → [A]
  • A done → pop A → Stack = []
  • Visit order: A → B → D → E → C → F

Characteristics

  • Traversal Order: Explores deeply along a path before backtracking.
  • Path Discovery: Useful for detecting cycles, topological sorting (DAGs), and connected components.
  • Data Structure: Uses a Stack explicitly or the call stack via recursion.
  • Applicability: Works on trees and general graphs (directed or undirected).
  • Time Complexity: O(V + E), where V = vertices and E = edges.
  • Space Complexity: O(V) in the worst case (stack depth/visited set).

DFS is widely used for exploring structure, discovering components, solving puzzles/mazes, and performing tasks like topological sorting and cycle detection, thanks to its depth-oriented strategy and simple mechanics.
